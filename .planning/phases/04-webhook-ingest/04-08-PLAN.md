<objective>
Wave 4A — formalize the webhook telemetry surface per CONTEXT D-22 (6 events) + D-23 (closed metadata whitelist). Plans 04 + 07 used inline `:telemetry.span/3` calls; this plan extracts them into named span helpers in `Mailglass.Webhook.Telemetry` (mirroring Phase 3's `Mailglass.Telemetry.send_span/2` pattern per PATTERNS.md lines 593-625) and refactors Plug + Reconciler to use them.

Purpose: Centralizing the helpers gives one place to enforce the D-23 whitelist (no `:ip`, no `:user_agent`, no `:raw_payload`, no header values), simplifies the Phase 6 LINT-02 `NoPiiInTelemetryMeta` Credo check (a single-module surface to lint), and gives adopters one module to attach telemetry handlers against. Also adds the three single-emit helpers (normalize, orphan, duplicate) that the Plug + Ingest module need to emit per-event signals to operator alerting.

Output: 1 new lib module + 2 modified lib modules + 1 new test file; mitigates T-04-07 (PII in telemetry — centralized whitelist enforcement); foundation for Phase 6 LINT-02 enforcement.

         | `[:mailglass, :webhook, :duplicate, :stop]` | single emit | `provider, event_type` |
         | `[:mailglass, :webhook, :reconcile, :start \\| :stop \\| :exception]` | full span | `tenant_id, scanned_count, linked_count, remaining_orphan_count, status` |

         **NEVER include in any metadata map:**

           * `:ip`, `:remote_ip`, `:user_agent`
           * `:to`, `:from`, `:subject`, `:body`, `:html_body`, `:headers`,
             `:recipient`, `:email`
           * `:raw_payload`, `:raw_body`

         Adopters wanting IP-based abuse triage attach their own handler
         on `[:mailglass, :webhook, :signature, :verify, :stop]` and pull
         `conn.remote_ip` from their own plug lineage (see
         `guides/webhooks.md`).

         Phase 6 LINT-02 (`NoPiiInTelemetryMeta`) lints THIS module + every
         caller against the forbidden-key set.
         """

         alias Mailglass.Telemetry

         @doc """
         Wrap the entire webhook ingest path in a `[:mailglass, :webhook, :ingest, *]` span.

         Stop metadata SHOULD include `:provider`, `:tenant_id`, `:status`,
         `:event_count`, `:duplicate`, `:delivery_id_matched`. NEVER include PII.
         """

         @spec ingest_span(map(), (-> result)) :: result when result: term()

           Telemetry.span([:mailglass, :webhook, :ingest], metadata, fun)
         end

         @doc """
         Wrap `Provider.verify!/3` in a `[:mailglass, :webhook, :signature, :verify, *]` span.

         Stop metadata SHOULD include `:provider, :status, :failure_reason`.
         """
         @doc since: "0.1.0"
         @spec verify_span(map(), (-> result)) :: result when result: term()
         def verify_span(metadata, fun) when is_map(metadata) and is_function(fun, 0) do
           Telemetry.span([:mailglass, :webhook, :signature, :verify], metadata, fun)
         end

         @doc """
         Single-emit per-event normalize signal (`[:mailglass, :webhook, :normalize, :stop]`).

         Metadata SHOULD include `:provider, :event_type, :mapped`. Alertable on
         sustained `mapped: false` rate (D-22).
         """
         @doc since: "0.1.0"
         @spec normalize_emit(map()) :: :ok
         def normalize_emit(metadata) when is_map(metadata) do
           Telemetry.execute([:mailglass, :webhook, :normalize, :stop], %{count: 1}, metadata)
         end

         @doc """
         Single-emit per-event orphan signal (`[:mailglass, :webhook, :orphan, :stop]`).

         Metadata SHOULD include `:provider, :event_type, :tenant_id, :age_seconds`.
         Fires once per normalized event that lands without a matching Delivery.
         """

         end

         @doc """
         Single-emit per-batch duplicate signal (`[:mailglass, :webhook, :duplicate, :stop]`).

         Metadata SHOULD include `:provider, :event_type`. Lets adopters

         """
