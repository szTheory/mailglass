# Phase 07.1 Plan 09: Post-Publish Smoke Workflow Summary

Created the post-publish smoke workflow that exercises the *published* Hex artifacts (not in-tree source) end-to-end — Hex.pm indexing, HexDocs build, Phoenix-host installer run, retraction check, and a tracker issue on failure.

## What Changed

- NEW file `.github/workflows/post-publish-smoke.yml` (394 lines, single commit `87206d3`).
- Three triggers per D-19: `workflow_run` on `publish-hex` completed, daily `cron: "0 12 * * *"`, and `workflow_dispatch` with optional `version` input.
- Six chained jobs per D-20: `cron-guard` → `wait-for-index` → `wait-for-hexdocs` → `consumer-install` → `retracted-check` → `notify-on-failure`.
- `cron-guard` enforces D-19 7-day window via `github.rest.repos.listReleases`; downstream jobs short-circuit via `if: needs.cron-guard.outputs.should_run == 'true'`.
- `wait-for-index` polls `mix hex.info` for both packages at 30s intervals up to 5 min (D-20 step 1).
- `wait-for-hexdocs` polls HexDocs URLs at 60s intervals up to 10 min (D-20 step 2).
- `consumer-install` uses Phoenix host per D-21: `mix archive.install hex phx_new --force` + `mix phx.new sandbox --module Sandbox --app sandbox --no-ecto --no-mailer --install`. Pins deps to `== ${VERSION}`, runs `mix mailglass.install --yes`, compiles with `--warnings-as-errors`, asserts no `UndefinedFunctionError` (literal G-1 regression on the published artifact), boots `mix phx.server &`, curls `http://localhost:4000/dev/mail/`, asserts HTTP 200, traps to kill server.
- `retracted-check` runs `mix hex.info <pkg> <ver>` for both packages and asserts output does NOT contain `Retired` (D-20 step 4).
- `notify-on-failure` mirrors `provider-live.yml` lines 52-127 — only label `publish-smoke-failed` and title `publish-smoke failure tracker` differ. Creates label if missing (color `B60205`), updates existing tracker issue or creates new one.
- All third-party Actions reuse existing repo SHA pins (`actions/github-script@60a0d83…`, `erlef/setup-beam@5304e04e…`) per D-34. No new pins introduced.
