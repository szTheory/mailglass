     groupName lives here per release-please-config.json: -->

From release-please-config.json (DO NOT MODIFY in this plan; reference only):
```json
"plugins": [
  {
    "type": "linked-versions",
    "groupName": "mailglass-sibling-group",
    "components": ["mailglass", "mailglass_admin"]
  }
]
```

The `groupName` produces a single tag of shape `mailglass-sibling-group-v<version>` (e.g. `mailglass-sibling-group-v0.1.0`) — see release-please linked-versions plugin docs cited in CONTEXT canonical_refs.

This is why both `mix.exs` files MUST set:
```elixir
source_ref_pattern: "mailglass-sibling-group-v%{version}"
```
The literal `%{version}` token is interpolated by ExDoc at doc-build time.
