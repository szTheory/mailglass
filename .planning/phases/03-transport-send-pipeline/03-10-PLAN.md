<objective>
Fix HI-01: MailerCase mutates global `:async_adapter` Application env on every
test setup, causing races between concurrent async tests. The `on_exit` unconditionally
restores to `:oban` even if the adopter had `:task_supervisor` configured at boot.

Fix: Guard the `Application.put_env` with `unless async? do ... end`. For async? == true
tests, `deliver_later/2` opts can carry `:async_adapter` (already supported per
outbound.ex:331 `Keyword.get(opts, :async_adapter)`). The `on_exit` snapshots the
pre-setup value and restores it instead of hard-coding `:oban`.

Also address the Oban :manual coverage gap: ensure tests tagged `@tag oban: :manual`
can exercise `assert_enqueued(worker: Mailglass.Outbound.Worker)` by confirming
the `oban_jobs` table is available in the TestRepo (or documenting the setup in
an oban_helpers.ex support module).

Purpose: Eliminate intermittent CI flakiness on async deliver_later tests and
prevent silent adopter config corruption from on_exit restores.

Output:
- test/support/mailer_case.ex — guarded async_adapter mutation + snapshot/restore
- test/support/oban_helpers.ex — Oban :manual setup helper (new file) if oban_jobs table needs explicit setup
</objective>

Target — guard and snapshot:
```elixir
# Capture the pre-setup value for faithful restore in on_exit.
prior_async_adapter = Application.get_env(:mailglass, :async_adapter)

cond do
  oban_tagged? and Code.ensure_loaded?(Oban) ->
    # ... Oban setup (unchanged) ...

  true ->
    # For async: false tests only — mutating global env is safe only when tests
    # are sequential. Async tests must use opts: [async_adapter: :task_supervisor]
    # passed to deliver_later/2 for isolation (already supported: outbound.ex:331).
    unless async? do
      Application.put_env(:mailglass, :async_adapter, :task_supervisor)
    end
    Mailglass.Adapters.Fake.set_shared(self())
end

on_exit(fn ->
  Mailglass.Adapters.Fake.checkin()
  Mailglass.Adapters.Fake.set_shared(nil)
  Mailglass.Clock.Frozen.unfreeze()
  # Restore whatever was set before this test ran (not unconditionally :oban).
  if prior_async_adapter do
    Application.put_env(:mailglass, :async_adapter, prior_async_adapter)
  else
    Application.delete_env(:mailglass, :async_adapter)
  end
  Ecto.Adapters.SQL.Sandbox.stop_owner(pid)
end)
```
